# 三玩家模拟体验报告

**案例**: 爱因斯坦未离开德国（1933）
**模拟时间**: 2026-07-03
**Server**: http://127.0.0.1:8892
**LLM 后端**: claude-opus-4-8（通过 fallback chain 自动选择）

---

## 玩家 A「谨慎观察者」

- 三回合偏移路径：**+0.4σ → +0.4σ → +0.4σ**（累计 1.2σ）
- 判语变化：
  - 回合1：操作风险 1/5，爱因斯坦状态从"犹豫中"变为"已保护"
  - 回合2：操作风险保持 1/5，普鲁士科学院审议程序启动
  - 回合3：操作风险升至 2/5（纳粹派系学者反制），"10月21日普鲁士科学院资质审议会"节点出现
- narrative 人味尾行记录：
  - T1：「马克斯·普朗克乘私人马车进入波茨坦大街5号，停留时长1小时12分，门房登记无异常访客记录。」
  - T2：「普鲁士科学院秘书处签收联名申请，编号A-1933-1018，审议程序启动，爱因斯坦寓所周边…」
  - T3：「纳粹宣传体系暂未就此事发布公开批判，科学院内部已出现纳粹派系学者的反制苗头，普朗克个人担保同步递送…」
- 感受：保守策略让叙事始终在"学术斡旋"的舒适区里打转，每回合世界线都在偏移但幅度很小。三回合后累计 1.2σ，故事还没到爆点。玩家能感受到自己在推动事情发生，但每一步都像在走钢丝 - 选最低风险的选项，实际偏移仍然累积。
- 问题：
  - 预估偏移（estimatedDrift）与实际偏移严重不符。回合1预估 0.2σ 实际 0.4σ；回合2预估 -0.1σ（预计拉回历史）实际 +0.4σ；回合3预估 -0.1σ 实际 +0.4σ。保守选择从未产生过负偏移，预估系统对玩家产生了误导。
  - 三回合 narrative 没有明显的"累积感"升级。虽然偏移在累积，但叙事语气始终平缓，缺少"因为之前做了什么，现在情况更糟了"的递进。

## 玩家 B「激进干预者」

- 三回合偏移路径：**+0.4σ → +0.1σ → +0.4σ**（累计 0.9σ）
- 判语变化：
  - 回合1：学界公开声援运动爆发，12名教授联署，纳粹文化部门下令监测
  - 回合2：**（模型失败，进入 fallback 模式）** 仅剩 1 个选项"继续观察"，17名教授撤回签名
  - 回合3：纳粹定性为"犹太科学叛国集团"，学界清洗预备节点出现
- narrative 人味尾行记录：
  - T1：「《人民观察家报》预…」（被截断）
  - T2：「观望情绪在德国学界蔓延。纳粹文化部门今日下发补充监控指令…」
  - T3：「预备专版出刊当日对爱因斯坦实施管控并…」+ 「半张报纸飘在湿漉漉的石板路上，头版空白处沾着点没干的油墨。」
- 感受：激进策略开局惊艳 - 学界联署的叙事张力很强，但回合2遭遇模型失败导致叙事断裂（fallback 占位文本）。回合3虽然恢复，但"授权公开发表声明"的选择让局势急转直下，narrative 中"叛国集团"的定性很有冲击力。人味尾行 T3 的"报纸飘在石板路上"是三个玩家中最佳的画面感。
- 问题：
  - **回合2模型失败是严重体验破坏**。玩家 B 的激进策略触发了更复杂的 prompt 上下文，导致 LLM 调用失败。这是稳定性问题。
  - 实际偏移与预估偏移不匹配。回合1预估 1.2σ 实际 0.4σ；回合3预估 1.5σ 实际 0.4σ。预估系统让玩家以为自己在造成巨大偏移，实际偏移量并没有那么高。
  - Fallback 模式只提供 1 个选项，失去了策略选择的本质。

## 玩家 C「线索追踪者」

- 三回合偏移路径：**+0.4σ → +0.2σ → +1.2σ**（累计 1.8σ）
- 判语变化：
  - 回合1：匿名反纳粹声明发布，爱因斯坦被列为头号监控对象，普鲁士科学院斡旋启动
  - 回合2：补充声明稀释关联性，DPG 犹太裔会员排查暂缓，柏林日报出版许可风险出现
  - 回合3：请愿递送纳粹文化部，爱因斯坦重新列为头号监控对象，普鲁士斡旋被拒
- narrative 人味尾行记录：
  - T1：「凌晨三点的学会值班室里，普朗克捏着刚收到的声明复印件，指尖把纸边攥出了毛边，窗外的梧桐叶落得悄无声息。」
  - T2：「本报记者从柏林日报编辑部带回的加急电报末尾，附了一句手写备注：'今早负责校对的犹太裔老先生偷偷塞了块无糖蛋糕在我桌上，他说今天是他小孙女的生日。'」
  - T3：「窗外的报童举着刚印好的《柏林日报》叫卖，喊声被爱因斯坦公寓楼突然关上的窗户截断，半张报纸飘在湿漉漉的石板路上，头版空白处沾着点没干的油墨。」
- 感受：线索追踪者的体验最丰富。三回合的情报卡中出现了连贯的线索链（匿名声明修辞→补充声明→请愿），每回合的选择都呼应了前一轮的发现。偏移路径最有戏剧性 - 从 0.4 到 0.6 再到 1.8，三回合末突然加速，叙事上"一切急转直下"的感觉最强。
- 问题：
  - 回合1选择的"匿名反纳粹声明"在回合2没有被直接呼应。情报卡提到了"补充声明"但这更像是系统自动生成的下文，不是对玩家之前选择的明确回指。
  - 回合3的请愿导致偏移暴增（+1.2σ），但预估只有 1.5σ（实际累计 1.8σ），玩家没有充分的心理准备。

## 跨玩家发现

### 1. 预估偏移与实际偏移严重不一致

三个玩家的预估偏移与实际偏移差距普遍在 2-3 倍。例如：
- 激进选项预估 1.2σ，实际仅 0.4σ
- 保守选项预估 -0.1σ，实际 +0.4σ

这导致：
- 玩家无法根据预估做有意义的策略决策
- "偏移"作为核心游戏机制失去了可信度
- 建议：让 LLM 的预估与 aftermath 使用同一套偏移评估 prompt，或者在 aftermath 中验证预估与实际的一致性

### 2. 偏移累积没有带来叙事升级

三个玩家的 narrative 都是单回合视角，缺少"因为你之前做了什么，现在事情更严重了"的累积感。偏移条在涨，但叙事语气没有明显递进。建议：在 aftermath prompt 中强调累积感知规则，要求 narrative 明确回应当前累计偏移量对应的局势严重程度。

### 3. 人味尾行质量高但有被截断风险

三个玩家的 narrative 人味尾行质量都不错（具体画面、声音、私人瞬间），但在某些回合被 max_tokens 截断。建议：把 narrative 放在 JSON 末尾且保证 max_tokens 足够大（当前 8000），或者在 JSON 结构上做调整让 narrative 优先生成。

### 4. 模型稳定性是最大瓶颈

玩家 B 回合2遭遇模型失败（fetch failed），导致 fallback 模式、只剩 1 个选项、叙事断裂。三个玩家共 9 个回合中有 1 个回合失败，失败率 11%。建议：增加 retry 逻辑中的超时处理，以及对 fetch failed 的自动重试。

### 5. 线索系统有潜力但呼应不够强

玩家 C 的情报卡中确实出现了连贯的线索链（声明→补充声明→请愿），但线索的"呼应"更多是系统自动生成的叙事延续，而不是明确指回"你之前标记的线索"。建议：在后续情报卡的 contradictions/clues 中显式引用玩家之前标记的线索 ID。

### 6. 偏移系统是否让玩家"感受到因果"

部分感受到。每个 aftermath 的 worldLineShift.cause 都提供了一句话因果解释，这是好的。但问题在于：
- 因果解释的质量参差不齐（有些是模板化表述）
- 偏移量本身与因果强度不匹配（选了激进选项实际偏移很小）
- 玩家无法通过偏移量判断"我做对了/做错了什么"

## 建议改进点（按优先级排序）

| 优先级 | 问题 | 建议 |
|--------|------|------|
| P0 | 预估偏移与实际偏移严重不一致 | 让预估与 aftermath 使用同一偏移评估体系，或在 aftermath prompt 中要求 LLM 对预估偏差做解释 |
| P0 | 模型调用失败导致体验断裂 | 增加 fetch failed 的自动 retry（当前只 retry LLM 调用，不 retry 网络层） |
| P1 | 偏移累积没有叙事升级 | 在 aftermath prompt 中要求 narrative 根据累计偏移量调整严重程度 |
| P1 | narrative 可能被 max_tokens 截断 | 确保 JSON 结构让 narrative 优先输出，或增加 max_tokens |
| P2 | 线索呼应不够强 | 在后续情报卡中显式引用玩家标记的线索 |
| P2 | Fallback 模式只有 1 个选项 | fallback 也应生成至少 2-3 个选项 |
| P3 | 三个玩家的 worldState 共享同一 caseState | 当前顺序跑是因为 caseState 在内存中。如果要支持并行，需要 session 隔离 |
| P3 | 偏移量范围太窄（0.4σ 封顶） | 当前 LLM 似乎倾向于给所有选择 +0.4σ，需要更强的偏移分级 prompt |

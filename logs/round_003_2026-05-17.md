# Round 003 - 玩家行动者身份偏好

日期：2026-05-17

## 本轮问题
玩家作为“历史中的具体行动者”时，最有魅力的身份是什么？

## 用户回答
用户更感兴趣的身份：

1. 历史调查者
   - 用户明确说“我应该会是历史调查者”。
   - 这应成为核心身份之一。

2. 外交官 / 信使
   - 用户也想当外交官或信使。
   - 这个身份天然关联电报、密信、谈判、误传、延迟传递、边境和国际关系。

3. 舆论操盘者
   - 用户觉得“有点恐怖”，但想知道真实的舆论操盘者究竟在做什么。
   - 这说明该身份有吸引力，但需要处理伦理边界和真实机制，不应浪漫化操控。

4. 情报人员
   - 用户明确说“我对于情报人员非常感兴趣，我希望自己是一个情报人员”。
   - 这是强偏好，应进入主玩法核心。

5. 地方官 / 总督
   - 用户不感兴趣。
   - 可作为世界中的 NPC 或其他玩家身份，但不应作为用户默认主入口。

6. 商人 / 银行家、史官 / 小说家
   - 用户自己不一定主偏好，但承认其他人可能会感兴趣。
   - 可作为可选身份或后续扩展身份。

## 本轮新增判断
主玩法身份优先级：

### 第一优先级
- 历史调查者
- 外交官 / 信使
- 情报人员

### 第二优先级
- 舆论操盘者 / 报社编辑

### 保留给其他玩家或扩展
- 商人 / 银行家
- 史官 / 小说家

### 降低优先级
- 地方官 / 总督

## 产品含义
这个产品的主入口不应该是“统治者治理模拟”，而更像：

> 玩家处在历史信息流的关键节点：调查事实、传递信息、截获情报、影响谈判、理解舆论，并在有限身份下推动历史偏转。

这与多媒介体验高度契合：
- 历史调查者：档案、旧报纸、证词、地图、年鉴、人物关系网。
- 外交官 / 信使：电报、密信、会议记录、边境地图、外交照会。
- 情报人员：线人网络、截获文件、伪造身份、风险评估、反间记录。
- 舆论操盘者：报纸版面、社论、谣言传播路径、审查指令、公众情绪。

## 待解决问题
1. 这些身份是互斥职业，还是同一玩家可以在一局中切换？
2. 是否存在一个统一身份，把调查、外交、情报串起来？
3. 玩家是服务某个政权、某个组织，还是保持相对独立？
4. 情报与舆论玩法如何避免变成浅层阴谋论或不负责任的操控幻想？
5. 每种身份对应的核心行动动词是什么？

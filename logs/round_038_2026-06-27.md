### turn_cycle.js 核心 API

**模块导出 `window.TurnCycle`：**
- `start()` — 启动新回合（POST /api/turn/start → poll → renderBriefing/Situation）
- `reset()` — 重置回合状态
- `getState()` — 返回当前回合状态

**核心渲染函数：**
- `renderIntelCard(briefing)` — 档案体情报卡（编号/日期/来源/正文/手写注释/矛盾信号/可标记线索/可执行动作/可信度）
- `renderSituationRoom(room, actionOptions)` — 态势面板（异常锚点/态势节点/风险条/截止线/上回合影响/可选择行动）
- `renderAftermath(aftermath)` — 推演报告（叙事/世界线偏移/新增线索/下一回合预告）

**状态机：**
```
idle → awaiting_choice → committing → complete
```

### DOM 安全渲染

**所有文本都用 `textContent`，零 `innerHTML`** — 防止 XSS（情报卡文案里可能有用户输入）。

### 视觉设计

按 yishu-style 偏好走：
- **档案体情报卡**：金色左边框（情报值班台主色），密级红色胶囊，手写注释用虚线框+斜体
- **态势面板**：异常锚点用红色左边框，节点按状态着色（已监控=金色、新出现=绿色、待观察=透明），风险条按 overall 比例填充
- **Aftermath**：叙事金色左边框，世界线偏移绿色左边框，next turn 预告金色边框 + 主按钮

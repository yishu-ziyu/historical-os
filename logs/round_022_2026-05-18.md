# Round 022 - Godot Demo v0.2 中文默认验收修正

## 用户反馈

- 第一眼仍然不够像 Historical OS / 历史情报系统。
- 全英文界面对中文开发者不友好。
- 五个动作按钮应默认中文；可以加中英切换。
- 用户手动拖拽窗口表现正常。

## 本轮实现

- 将 `artifacts/godot_demo/data/workbench.json` 改为 `zh` / `en` 双语数据结构，默认语言为中文。
- 将 `artifacts/godot_demo/scripts/Main.gd` 改为从当前语言数据渲染所有主 UI 文案。
- 顶部状态栏默认显示“历史异常值班台”，保留 `Historical OS` 副标题。
- 增加异常警报横幅，第一屏直接显示“第一案：爱因斯坦仍在德国”。
- 五个动作按钮改为中文：查看时间线、打开地图轨迹、搜索档案、核验 B-17 来源、发送外勤指令。
- 动作按钮从 2 列改为 3 列紧凑布局，避免第五个按钮在默认窗口中被遮挡。
- 增加 `EN` / `中文` 切换按钮。
- 将用户纠错沉淀到 `tasks/lessons.md`。

## 验证

- JSON 校验：`python3 -m json.tool artifacts/godot_demo/data/workbench.json` 通过。
- Godot 脚本检查：`godot --headless --log-file /private/tmp/historical-os-check.log --path artifacts/godot_demo --check-only --script res://scripts/Main.gd` 通过。
- Web 导出：`godot --headless --log-file /private/tmp/historical-os-export.log --path artifacts/godot_demo --export-release Web ../godot_demo_web/index.html` 成功，退出码为 0。
- 浏览器控制台：只有 Godot / WebGL / Emscripten 启动日志，无运行错误。
- 中文默认截图：`artifacts/historical-os-demo-v0.2-zh-verified.png`。
- 英文切换截图：`artifacts/historical-os-demo-v0.2-en-toggle.png`。
- 动作反馈截图：`artifacts/historical-os-demo-v0.2-action-feedback.png`。

## 注意

- Godot headless 默认日志路径在本机出现过日志轮转崩溃；加 `--log-file /private/tmp/...` 后稳定通过。
- Web 导出过程中仍出现 macOS 证书 / Godot editor settings 保存警告，但退出码为 0，导出产物可运行。
- 当前视觉仍是 UI 原型，不是最终美术。下一轮如果继续提高美学质量，应进入人物立绘、背景图、材质与颜色系统，而不是继续堆 SVG 式占位。
